1.2 HashMap

HashMap 是一个散列表，它存储的内容是键值对(key-value)映射。 在HashMap中通过get()来获取value，通过put()来插入value，ContainsKey()用来检验对象是否已经存在。LinkedHashMap保存了记录的插入顺序，在使用Iterator遍历的时候，保证了元素迭代的顺序。而HashMap遍历的时候获取的数据完全是随机的。
